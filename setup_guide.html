1

Go to Google Cloud Console

Open this link in a new tab:

Open Google Cloud Console

You'll need to sign in with your Google account.

2

Create a New Project

  • Click "Select a project" at the top of the page
  • Click "NEW PROJECT" button
  • Project Name: YouTube Content Detection
  • Click "CREATE" button
  • Wait a few seconds for it to create
3

Enable YouTube Data API v3

  • In the search bar at the top, type: YouTube Data API v3
  • Click on "YouTube Data API v3" from results
  • Click the big blue "ENABLE" button
  • Wait for it to enable (takes 5-10 seconds)
4

Create API Key

  • Click "CREATE CREDENTIALS" button
  • Select "API Key" option
  • COPY the generated API key (it looks like: AIzaSyDxxxxxxxxx...)
  • Click "CLOSE"
5

Add API Key to config.py

Open this file in your project:

C:\Users\reddy\OneDrive\Desktop\project file\config.py

Replace this line:

YOUTUBE_API_KEY = 'YOUR_YOUTUBE_API_KEY_HERE'

With your actual API key:

YOUTUBE_API_KEY = 'AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxx' # Paste your key here
6

Restart Django Server

In your terminal/PowerShell:

  • Press CTRL + C to stop the server
  • Run the server again:
python manage.py runserver
